# 头部玩家与关键变量（看格局与破局点）

搞清商业模式与市场大盘后，看具体谁在赚大钱、正在发生什么变化。来源：方法论02 §3 + Porter 五力 + PESTEL。

目录：
- 头部玩家三看 + 五维打分
- 行业集中度与壁垒
- Porter 五力（系统检查清单）
- PESTEL 与关键变量
- 动态看边界与替代

---

## 头部玩家：研究行业最好的样本（三看）

头部公司的动作就是行业的方向。
- **看壁垒**：它们怎么赢的？产品 / 渠道 / 成本 / 品牌心智？头部赢的方式 = 行业关键成功要素。
- **看投入**：资源往哪投？组建什么部门、招什么人、死磕什么技术、拓什么业务——信号比报告真实。看它们干什么、干出什么结果、做了什么转变，而非听它们说什么。
- **看舍弃**：放弃了什么？头部放弃的方向往往是被验证走不通或性价比太低；但要辩证——头部走不通的，对你的体量也许性价比更高。

**五维打分**（品牌力 / 产品力 / 渠道力 / 营销力 / 资本力，各 1-5 分）：标出"我们现在在哪、要去哪"，路径就清晰了。

---

## 行业集中度与壁垒

- **集中度**：看 CR4（前四市占之和）/ CR8。高=头部壁垒强、资源心智已被占、新玩家难入；低=分散、有整合突围机会。
- **六类壁垒**：品牌 / 规模 / 技术 / 渠道 / 人才 / 用户粘性。壁垒越高进入越难，但站稳后就是护城河。

## Porter 五力（系统检查清单，别漏项）

借 Porter 五力系统核查行业竞争烈度与盈利能力：
1. 现有竞争者的竞争强度
2. 新进入者威胁
3. 供应商议价能力
4. 买方议价能力
5. 替代品威胁

五力共同决定行业平均利润水平。与上面的"利润集中在哪/壁垒/集中度"交叉印证。

## PESTEL 与关键变量（找改变格局的变量）

宏观层用 PESTEL 扫一遍（政治/经济/社会/技术/环境/法律），筛出真正影响本行业的关键变量。方法论02 重点关注三类：
- **技术变量**：有无新技术改变成本结构与用户体验（电池之于新能源车、AI 之于内容业）——常是洗牌最大驱动力。
- **政策变量**：鼓励/限制、放开/收紧，直接决定天花板与利润分配（双减之于 K12、补贴之于新能源车）。
- **供需变量**：用户趋势在变什么？供给端有无新玩家搅局？

可做"关键变量判断表"：把变量按 1-5 重要度排序。

---

## 动态看边界与替代（对 Porter 的修正）

Porter 五力假设行业边界固定，会忽略快速技术突变与跨界合作——AI 时代尤其明显。所以判断格局时，**别把行业边界看死**：今天的替代品、邻近赛道的玩家、新技术催生的新解法，都可能重画边界。这与"替代品威胁""技术变量"一脉相承。
