# 切页与页面文案

把每章内容切成"一页一观点"，并让每页有说服力。来源：方法论15（#37/#25/#39/#43/#69）+ 文案公式（PAS/FAB/BAB/Cost-of-Inaction）+ 情绪弧（Duarte）。

## 一页一观点：怎么切（方法论15 #37）

"一页一个观点，遵循'现象+结论'或'论点+论据'的结构，让每一页只传递一个清晰、明确的信息。"

操作：
1. 把一章的交汇判断列出来，每个独立判断 → 一页。
2. 每页配 ≤4 条论据。**论据超过 4 条 = 信号：这页其实是两个观点，拆成两页**（讲不完分两页，宁可多一页）。
3. 每条论据挂分析卡 `evidence_refs`。没有出处支撑的判断 → 标为假设或砍掉。
4. 能删则删（#80）：与本页判断无关的论据，再有趣也不放。

## 判断句标题（governing_thought）

标题是这页的**结论**，写成完整判断句，读者扫标题就能获得论证主线。

- ❌ 话题词："品牌定位" / "竞品分析" / "市场现状"
- ✅ 判断句："LUMA 应占据'日常可及的专业精品'定位" / "竞品在高质平价与匠人叙事间留下中间带" / "市场增长正从开店红利切到复购红利"

每页先写核心句（#100），再填论据。

## 页面文案公式（按页型选，让论据有说服力）

| 页型 | 公式 | 模板 |
|---|---|---|
| 问题/痛点页 | PAS | "[痛点]？每[周期]，[代价]。[方案]解决它。" |
| 代价/紧迫页 | Cost of Inaction | "不[行动]，你每[周期]在损失[量]。" |
| 方案/转变页 | Before-After-Bridge | "[之前痛点]。[之后理想态]。[你的方案]是桥。" |
| 优势/能力页 | FAB | "[能力]让你[优势]，于是你能[利益]。" |
| 对比/前后页 | 前后对比图（#46） | 之前的痛点 vs 达成的效果，感性+理性双呈现 |

标题模式可借力：对比型"[旧方式]已死，认识[新方式]"、"从[痛点]到[利益]"；但避免空洞口号（见 writing-discipline）。

## 情绪弧（方案不只是逻辑，#43 情绪价值也是价值）

一份方案应有情绪走向，不是平的。典型弧：
`好奇 → 焦虑/不满(摆出冲突) → 希望(给出方向) → 信任(证据支撑) → 笃定/紧迫(走向行动)`

Duarte Sparkline 技巧：在"现状痛点(What Is)"与"理想未来(What Could Be)"之间**交替**，在 1/3、2/3 处制造张力峰值。这与 SCQA 的"现状-冲突"、双轴的"层层剥开"是同一套节奏。

## 策略表达有 Power（方法论15 #25/#39）

- 精准、有力、一针见血。多用短句、多用动词，少用形容词，避免冗余和歧义。
- 借丘吉尔演讲逻辑（#39）：强有力的开头 + 只讲一个主题 + 简单语言 + 留下画面感 + 戏剧性结尾。
- 沟通效率极致化（#42）：核心主张能否让客户在电梯里 30 秒讲清给他老板？不能就说明还不够简单有力。
