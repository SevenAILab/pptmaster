# 从分析到聚焦（自身分析的落点）

分析是过程，决策才是目的。自身分析完要落到行动，否则就是"正确的废话"。来源：方法论04 §4。

目录：
- 落地四步
- 短板的三种处理
- 落到分析卡

---

## 落地四步：找方向 → 做聚焦 → 敢舍弃 → 形成 SOP

1. **找方向**：先做正确的事，再正确地做事。正确的事三要素 = **用户要 + 竞争无 + 壁垒强**。竞争有空白的两种逻辑：完全空白（先发优势）/ 大的看不上、小的跟不上。
2. **做聚焦**：方向定了资源要跟上——以一当十而非以十当一。多数公司的问题不是没优势，而是资源分散，每个点都不痛不痒。把一个点做到极致 > 把十个点做成平庸。
3. **敢舍弃**：不做什么和做什么同等重要。聚焦的前提是舍弃，别让低投产比的事分散资源。
4. **形成 SOP**：拆目标 → 拆动作（谁做/做什么/何时交付）→ 定标准（优秀/良好/合格）→ 定流程（上一步产出=下一步输入）。

## 短板的三种处理（自身分析绕不开短板）

- **必须补**：对核心优势有影响的致命短板（产品质量不稳、品牌调性混乱）——直接断根基，必须补。
- **外包补**：可通过合作/外包解决的（不擅长物流就找第三方）——不必亲自补，资源放最擅长处。
- **绕开**：模式里的次要矛盾（拼多多早期不做物流极致，因用户对价格远比对速度敏感）——这不是短板，是战略选择。

## 落到分析卡

self-analysis 的卡最终要服务"资源往哪投、什么该做、什么不该做"：
- 优势卡 → `implication`：这个真优势该如何聚焦放大。
- 劣势/短板卡 → `implication`：必须补 / 外包 / 还是绕开。
- 方向卡 → `implication`：基于"用户要+竞争无+壁垒强"，建议聚焦的那个点。
