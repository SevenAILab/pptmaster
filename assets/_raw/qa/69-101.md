# 101

095什么是MVP？
最小可行产品（MVP Minimum Viable Product）是以最低成本、最快速
度推出，仅包含核心功能的早期产品版本，其核心目标在于快速验证市场假
设、收集用户反馈并指导后续迭代，
避免资源浪费。极简定义：以最小成本
验证产品核心价值的原型版本。
