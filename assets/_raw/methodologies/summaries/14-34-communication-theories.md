# 策略人必备的 34 个传播学理论 · 摘要

## 核心方法

传播不是简单输出信息，而是影响注意力、理解方式、社会认同和行为决策。传播学理论可以帮助策略 Agent 判断：说什么、对谁说、通过什么媒介说、如何被理解、如何被扩散。
