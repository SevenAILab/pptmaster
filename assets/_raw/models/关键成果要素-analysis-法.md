# 关键成果要素分析法

127. 关键成功要素分析法
适用场景：企业战略、品牌策略、个人决策
理论来源：哈佛大学教授 William Zani
关键成功要素 (KSF，Key Success Factors) 是一种管理工具，用于识别和定义对于组织或项目成功至关重要的几个关键领域或因素。
这些因素是组织必须做得好的事情，以实现其目标和战略。KSF并非所有重要的事项，而是那些对成功影响最大的因素。KSF具有高度优先级，是组织资源配置和管理重点的依据。
