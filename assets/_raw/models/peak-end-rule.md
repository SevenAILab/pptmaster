# 峰终定律

132. 峰终定律

适用场景：产品用户体验优化
理论来源：丹尼尔·卡尼曼
峰终定律是由诺贝尔经济学奖获得者、心理学家丹尼尔·卡尼曼提出的一种认知偏差理论。它表明，人们对一段体验的记忆主要取决于“峰值时刻”（即最强烈的情绪体验，无论是正面还是负面）和“结束时刻”的感受，而不是整个过程的平均体验。
比如航空公司在航班中关注“峰值”时刻（如空姐热情服务、舒适的机舱环境）和“终值”时刻（如便捷的下机流程、友好的道别）。
