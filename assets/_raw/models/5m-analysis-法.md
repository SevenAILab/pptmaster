# 5M分析法

129. 5M分析法
适用场景：质量管理、流程优化
理论来源：/
Man（人员）：与人相关的因素。检查是否是操作人员的技能不足或操作失误导致的问题。比如员工是否经过充分的培训，操作是否符合标准作业程序。
Machine（机器）：设备、工具等硬件因素。检查生产设备是否存在故障或老化，是否需要进行维修或升级。
Material（材料）：原材料、零件等投入要素。检查原材料是否合格，是否存在材料采购、存储或运输上的问题。
Method（方法）：工作程序、操作规范等。检查现有的生产工艺和流程是否合理，是否有不必要的步骤或潜在的改进空间。
Measurement（测量）：检验标准、计量方法等。检查质量检测手段是否有效，是否有合适的测量工具和标准，以及是否及时反馈问题。
