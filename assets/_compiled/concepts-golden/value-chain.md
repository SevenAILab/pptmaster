# Value Chain · 价值链

## 定义

价值链是把企业或产业中创造价值的活动拆成一组相互连接的环节,用来识别价值增值位置、成本结构、利润池和竞争优势来源。

Michael Porter 在 1985 年提出企业价值链框架,将企业活动分为基本活动和支持性活动。Seven 247 书在行业分析里采用更适合品牌策略的产业链视角: 产品从原材料到最终消费者手中会经历上游、中游、下游一系列增值环节。理解价值链,不只是知道谁在哪个环节,而是看清钱、价值、议价权和壁垒如何流动。

在 PPTAgent 中,Value-Chain 主要用于行业分析和定位前置判断: 品牌到底处在链条哪里,哪个环节赚钱,哪个环节卡脖子,未来是否要向上游或下游延伸。

## 适用场景

- **行业分析必检**: Sub-Agent ② 输出产业链结构、利润池和关键环节
- **竞品壁垒判断**: Sub-Agent ③ 分析竞品优势来自研发、供应链、渠道还是服务
- **品牌生态位选择**: Sub-Agent ④ 判断品牌应做制造型、渠道型、内容型还是服务型定位
- **供应链/渠道风险识别**: 看清上下游议价能力和依赖关系
- **不适合**: 只画一条上中下游箭头；如果不分析利润、权力和机会,价值链只是流程图

## 使用步骤

### 第 1 步 · 明确分析边界

先定义行业或品类边界,例如 "中国线上手工皂行业" 或 "创作者影像配件行业"。边界不同,链条不同。

输出标准: 行业范围 + 地域范围 + 目标客户 + 主要替代方案。

### 第 2 步 · 绘制上中下游链条

把链条拆成上游供应端、中游生产/研发端、下游渠道/用户端。必要时加入物流、服务、内容、数据和平台环节。

输出标准:

| 环节 | 典型角色 | 关键活动 | 代表玩家 |
|---|---|---|---|
| 上游 | 原料/技术/设备 | 提供生产要素 | ... |
| 中游 | 研发/制造/品牌 | 创造产品价值 | ... |
| 下游 | 渠道/零售/服务 | 触达与转化用户 | ... |

### 第 3 步 · 标注价值和利润分配

分析每个环节的收入规模、毛利率、成本压力和利润池。不要只看谁离用户近,还要看谁能拿走利润。

输出标准: 高利润环节、低利润环节、利润转移趋势。

### 第 4 步 · 判断议价权和卡点

看谁掌握稀缺资源、技术、渠道、用户关系或标准。应收账款、应付账款、毛利率、交付稳定性都可作为判断信号。

输出标准: 关键控制点 + 被卡风险 + 可替代性判断。

### 第 5 步 · 定位自己的生态位

判断品牌现在处于哪个环节,优势在哪里,短板在哪里。早期品牌不一定要控制全链条,但必须知道自己依赖谁、被谁影响。

输出标准: 当前生态位 + 可守护优势 + 主要依赖。

### 第 6 步 · 提出延伸或协同机会

思考是否向上游延伸以控制原料/技术,向下游延伸以直接触达用户,或与关键伙伴建立协同。

输出标准: 3 个战略机会点 + 对应风险。

## 输入输出示例

### 示例 1 · 植愈坊线上手工皂价值链 (247 书贯穿教学案例)

> **案例可追溯报告**: 植愈坊是 Seven 247 书全书贯穿的**教学性虚拟案例** (非真实商业体)。具体出处行号待 Codex 后续按 P0-4 模板填充, 当前仅标注教学属性。**禁止包装成"真实品牌成功案例"**。

背景: 植愈坊计划从线上天然手工皂切入,初期依赖小作坊生产和社交电商获客。

| 环节 | 角色 | 判断 |
|---|---|---|
| 上游 | 油脂、精油、碱、包材供应商 | 基础原料议价弱,高品质精油和定制包材会影响差异化 |
| 中游 | 研发、设计、小作坊生产/OEM | 配方、设计、品控是核心价值创造环节 |
| 下游 | 小红书、微信私域、淘宝、物流、消费者 | 内容和信任决定转化,平台规则影响获客成本 |

战略启发:

```text
短期: 控制关键精油供应和配方品控,在下游用小红书/KOC 建立信任。
中期: 将小作坊生产转为可标准化小批量生产,补齐合规和产能。
长期: 向上游建立可追溯原料故事,向下游沉淀私域会员关系。
```

**关键洞察**: 植愈坊真正的壁垒不在"会做皂",而在配方审美、原料信任和用户关系三段连接。

### 示例 2 · 苹果价值链 (Tier 2 教科书案例)

> **案例可追溯报告**: 行业公认经典案例 (业内通用价值链分析对象)。本示例**只描述 Porter Value Chain 9 类活动的应用结构**, 不引用具体成本数据/利润率/内部财务数据。苹果"设计美国/制造亚洲"的全球价值链布局是业内公开常识。

Porter 价值链 (1985) 把企业活动分为"主要活动"和"支持活动"两组共 9 类。苹果是全球价值链布局的经典案例:

| 价值链类型 | 9 类活动 | 苹果布局 |
|---|---|---|
| **主要活动** | 入境物流 | 全球供应商: 高通 (基带) / TSMC (芯片) / 三星 (屏幕) / LG (电池) |
| | 生产运营 | 制造在亚洲: 富士康/和硕 (中国/印度/越南) |
| | 出境物流 | 全球分发: Apple Store + 授权经销商 + 直营网店 |
| | 营销销售 | 品牌+发布会+精品店全球统一 |
| | 售后服务 | Genius Bar + AppleCare 全球服务网 |
| **支持活动** | 采购 | 全球零部件采购优势 (规模 + 长期合作) |
| | 技术研发 | **核心 R&D 在加州 Cupertino** (设计/芯片/软件/AI) |
| | 人力资源 | 全球招聘核心人才, 美国总部决策 |
| | 企业基础 | 美国上市 + 全球税务架构 + IP 资产 |

**关键洞察**: 苹果价值链的**战略洞察是"高附加值环节在本地, 低附加值环节全球外包"** — 设计/品牌/IP 在美国 (附加值最高), 制造在亚洲 (成本最低)。**这是 Porter 价值链最经典的"微笑曲线"实践** — 价值链两端 (R&D 和品牌+服务) 利润率最高, 中间制造环节利润最低。**品牌策略人做价值链分析时, 要找到"高附加值环节"并强化它**, 而不是均匀投入所有环节。

## 常见误用

1. **画链条不看价值**: 只列上游中游下游 → 不知道钱在哪、壁垒在哪
2. **把产业链等同供应链**: 只看采购和生产 → 忽略渠道、用户关系和服务价值
3. **忽略利润池变化**: 过去赚钱的环节可能正在被平台或新技术挤压
4. **只看头部玩家**: 不看中小玩家和替代方案 → 容易误判真实竞争
5. **不分析议价权**: 不知道谁能压价、谁被卡脖子 → 战略风险被低估
6. **盲目全链条控制**: 小品牌想什么都自己做 → 资源被拖垮
7. **没有转化为定位**: 价值链分析完不影响品牌定位和商业模式 → 分析停在报告里
