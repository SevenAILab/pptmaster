# 上市策略 · GTM

## 定义

GTM (Go-to-Market) 是把产品或品牌推向市场的一套进入策略,用于明确首批用户、上市主张、渠道组合、传播节奏、销售路径、预算分配和复盘指标。

本概念在 Phase 1 中来自 MAP-STRUCTURE 的传播层与增长层连接: 它把品牌定位、产品卖点、渠道选择和增长动作串成一段可执行的上市路径。它缺少 Seven 247 书中的独立专章,但本地材料在竞品分析、新品上市、渠道选择和年度规划中反复出现相关动作: 上市节奏、蓄水期/引爆期/长尾期、预算分配、平台组合、内容策略和商业结果。

一句话: GTM 不是 "发布一下",而是定义谁先买、为什么买、在哪里买、怎么被说服、如何验证成功。

## 适用场景

- **新品上市规划**: Sub-Agent ⑥ 将产品、价格、渠道、传播和预算排成落地节奏
- **竞品上市复盘**: Sub-Agent ③ 分析对手从 0-1、1-10 的打法
- **品牌冷启动**: 明确原点人群、首批渠道和种草路径
- **定位落地**: Sub-Agent ④ 将定位语转成首轮市场叙事
- **不适合**: 产品价值和目标人群还没定义清楚时硬排投放计划；GTM 依赖前置策略

## 使用步骤

### 第 1 步 · 明确上市目标

先判断这次 GTM 是验证需求、建立认知、获取首批用户、冲销量,还是打开新渠道。

输出标准: 1 个主目标 + 2-3 个验证指标。

### 第 2 步 · 定义首批人群

选择最容易理解价值、最愿意尝试、最能形成反馈和传播的原点人群。

输出标准: 首批人群画像、关键任务、购买阻力。

### 第 3 步 · 提炼上市主张

把定位、产品卖点、RTB 和使用场景压缩成一套上市信息: 核心说法、证据、首发利益点。

输出标准: 上市核心信息 + 3 条 RTB + 首发利益点。

### 第 4 步 · 设计渠道和内容组合

选择触达人群的核心平台、内容形式、KOL/KOC、搜索/信息流/私域/电商承接路径。

输出标准: 渠道分工表和内容主题表。

### 第 5 步 · 排上市节奏

拆成蓄水期、发布期、引爆期、长尾期或验证期、放量期、复盘期,并分配动作和预算。

输出标准: GTM 时间线 + 预算/资源分配。

### 第 6 步 · 建复盘机制

跟踪认知、互动、转化、复购、渠道 ROI、用户反馈和问题闭环。

输出标准: 上市复盘看板 + 下一轮优化动作。

## 输入输出示例

### 示例 1 · 植愈坊新品上市 GTM (247 书贯穿教学案例)

> **案例可追溯报告**: 植愈坊是 Seven 247 书全书贯穿的**教学性虚拟案例** (非真实商业体)。具体出处行号待 Codex 后续按 P0-4 模板填充, 当前仅标注教学属性。**禁止包装成"真实品牌成功案例"**。

背景: 植愈坊推出 "晚安疗愈皂",希望验证情绪疗愈洁肤定位。

| 模块 | 输出 |
|---|---|
| 上市目标 | 验证睡前洁肤疗愈场景,获取 300 名首批试用用户 |
| 首批人群 | 25-35 岁压力型女性,敏感肌/成分党,重视睡前仪式 |
| 上市主张 | 把洗脸变成睡前 3 分钟疗愈仪式 |
| 渠道组合 | 小红书种草 + 私域试用群 + 淘宝承接 + 芳疗师内容背书 |
| 节奏 | 2 周蓄水种草,1 周试用发布,4 周反馈复盘 |
| 指标 | 笔记收藏率、试用申请率、复购意愿、敏感肌反馈 |

**关键洞察**: 植愈坊的 GTM 应先验证场景和信任,不要一开始用大促逻辑冲销量。

### 示例 2 · 假设智能家居品牌 X 新品 GTM (Tier 3 抽象示例)

> **案例可追溯报告**: 抽象演示案例 (非真实品牌, 用于演示 GTM 模块化结构). 任何与真实品牌的相似性纯属巧合。本示例不应被引用为"真实案例佐证"。

背景: 假设一家智能家居品牌 X 发布新一代智能灯具, 借家居装修旺季窗口上市:

| 模块 | 输出 |
|---|---|
| 上市目标 | 抢占"智能灯具升级"心智, 成为装修期首选 |
| 首批人群 | 半年内有装修计划的城市新中产 |
| 上市主张 | "一灯一场景, 装修期一次到位" |
| 渠道组合 | 装修平台合作 + 家居 KOC 实拍 + 官方旗舰 + 线下体验 |
| 节奏 | 装修旺季前 30 天预热, 旺季开始开售, 持续 90 天内容 |
| 指标 | 装修平台关联曝光、套装购买率、复购率、客单价 |

**关键洞察**: GTM 的**窗口型策略** — 不是"产品好就上市", 而是**找目标人群的"自然窗口"** (此例的装修旺季) + 用一套联动节奏抢占心智, 比"全年线性投放"效率高数倍。

## 常见误用

1. **把 GTM 当发布会排期**: 只有时间表,没有首批人群和购买理由 → 发布热闹但卖不动
2. **定位和上市脱节**: 战略页讲差异化,上市页讲促销 → 用户记不住品牌价值
3. **渠道越多越好**: 每个平台都做一点 → 预算被摊薄,没有核心阵地
4. **只看曝光不看承接**: 种草内容很多,购买路径和客服不顺 → 流量流失
5. **忽略竞品上市节奏**: 不知道对手如何蓄水和爆破 → 复用不到有效经验
6. **预算不跟目标匹配**: 验证期用放量预算,或放量期预算不足 → 节奏失真
7. **没有复盘指标**: 上市结束只看销量 → 不知道人群、信息、渠道哪个环节有效
