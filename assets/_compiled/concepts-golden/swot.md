# SWOT 分析

## 定义

SWOT 是一个战略分析与决策工具,把与品牌/业务相关的内外部因素归纳到四个象限:

- **S - 优势 (Strengths)**: 内部有利因素 — 你擅长什么、有什么独特资源
- **W - 劣势 (Weaknesses)**: 内部不利因素 — 你欠缺什么、哪些方面不如对手
- **O - 机会 (Opportunities)**: 外部环境有利因素 — 市场新需求、政策利好、对手失误
- **T - 威胁 (Threats)**: 外部环境不利因素 — 竞争加剧、替代品、政策收紧、需求转移

它的真正价值不在于"画格子",而在于强迫团队做**系统性思考 + 内外部因素关联匹配**,从而得到具体可执行的战略方向。理论来源: 管理学教授 Heinz Weihrich (1980s)。

## 适用场景

- **品牌战略起点**: 在做品牌定位/年度规划前,先用 SWOT 把家底盘一遍
- **竞品对照分析**: 自己做一份 SWOT, 同时对每个主要竞品也做一份, 找差异化机会
- **战略选项生成**: 用 TOWS 策略矩阵 (SO/ST/WO/WT) 从分析跳到行动
- **风险预判**: 在 T 象限提前列出潜在威胁, 配套准备应对预案
- **不适合**: 单纯做罗列堆砌而不做交叉策略 (退化为"漂亮的格子" — 详见"常见误用")

## 使用步骤

### 第 1 步 · 找内部优势 (S) — 见 247 书 §5.1

核心优势必须同时满足三个条件: **相对的 (比对手强) + 有价值的 (客户在意) + 难模仿的 (有壁垒)**。

从三个维度梳理:
- **资源**: 有形 (资金/原材料/专利/实体店/私域流量) + 无形 (创始人声誉/品牌文化/早期用户/合作伙伴)
- **技术**: 独特生产工艺 / 研发能力 / 平台技术 / 独家数据模型
- **能力**: 团队能力 (经验/创意/运营/服务/学习) + 组织能力 (决策/供应链/内容/成本)

⚠️ "我们团队很努力" 不是核心优势 (对手也很努力)。**优点 ≠ 核心优势**。

### 第 2 步 · 找内部劣势与风险 (W) — 见 247 书 §5.2

诚实地列出限制目标实现的内部因素:
- **资源缺乏**: 资金/人才/技术/供应链/品牌知名度
- **能力不足**: 营销/运营效率/研发/成本控制/客户服务
- **组织问题**: 沟通/决策速度/战略方向

同时列出内部相关的潜在风险: 运营风险 / 财务风险 / 管理风险 / 依赖风险 (过度依赖单一供应商/客户/渠道/核心成员)。

### 第 3 步 · 扫描外部机会与威胁 (O / T) — 引上游 ②③ 输出

O / T 通常来自:
- **市场趋势**: 消费习惯变化、新兴需求 / 偏好转移、需求下降
- **竞争格局**: 竞争者退出、市场整合 / 新进入者、价格战
- **技术变革**: 新技术应用 / 替代品出现、品类衰退
- **政策环境**: 利好法规、政策扶持 / 监管收紧

如果有 Sub-Agent ② industry_analysis 和 ③ competitor_analysis 的上游输出, 优先取自这两个 Agent 的结论, 不要重复造轮。

### 第 4 步 · 构建 SWOT 矩阵

把 S/W/O/T 各 3-6 个关键点列成一个 2×2 表格 (左上 S / 右上 W / 左下 O / 右下 T 是常见画法, 也可上下分内外)。

每个点要求:
- 一句话讲清, ≤ 25 字
- 用编号 (S1/S2/...) 方便后续策略矩阵引用
- 关键点要有 evidence (数据 / 客户反馈 / 对标分析), 不要凭直觉

### 第 5 步 · TOWS 策略矩阵 — 从分析到行动 (见 247 书 §5.4, 关键步骤)

四种战略组合,**不是相互排斥, 一个好战略通常包含多种组合**:

| 策略 | 公式 | 思路 |
|---|---|---|
| **SO 增长型** (Maxi-Maxi) | 优势 × 机会 | 用我的核心优势抓住外部机会,主攻方向 |
| **ST 防御型** (Maxi-Mini) | 优势 × 威胁 | 用优势规避或减轻外部威胁,建立壁垒 |
| **WO 扭转型** (Mini-Maxi) | 劣势 × 机会 | 利用外部机会弥补内部劣势 (找合作/利用红利) |
| **WT 收缩型** (Mini-Mini) | 劣势 × 威胁 | 在叠加场景下控损 (收缩范围/控成本/转型) |

战略权重一般是: **SO 主攻 + ST/WO 辅助 + WT 兜底**。

### 第 6 步 · 输出 3-5 个战略优先事项

矩阵不是终点。最后必须落到 3-5 个未来 3-6 个月的具体行动 (e.g. "完成 MVP / 获取 100 种子用户 / 完成 xx 融资")。这才是 SWOT 的真正交付物。

## 输入输出示例

### 示例 1 · 小米创立初期 SWOT (来自 247 书 §5.3)

> **案例可追溯报告**: 真实公知品牌。本示例 4 象限内容 (S1-S4 / W1-W4 / O1-O3 / T1-T3) **完全来自** `assets/_raw/books/0to1-brand/ch05-self-swot.md` L362-376 原文 (Seven 在 247 书 §5.3"全景透视"章节给出的小米 SWOT 经典案例), 未做任何 LLM 改写。"硬件+软件+互联网服务三位一体"战略演进同样来自该原文 L374-376。

```
S 优势:                        W 劣势:
  S1 互联网基因                  W1 品牌认知度低
  S2 极致成本控制能力             W2 硬件经验缺乏
  S3 社区运营经验                 W3 供应链控制弱
  S4 创始团队互联网背景            W4 专利储备少

O 机会:                        T 威胁:
  O1 智能手机爆发增长              T1 苹果/三星强势
  O2 传统手机品牌价格高            T2 国产品牌竞争
  O3 电商渠道崛起                  T3 专利诉讼风险
```

**战略输出 (SO 主攻)**: "硬件 + 软件 + 互联网服务" 三位一体 + 极致性价比 + 电商直营。

**演进**: 随发展不断更新 SWOT, 战略从单一手机扩展到 IoT 生态系统, 从电商专营拓展线下渠道, 从中国市场拓展全球。

### 示例 2 · 植愈坊 (247 书贯穿教学案例, ch05 §5.1-5.4 完整推演)

> **案例可追溯报告**: 植愈坊是 Seven 247 书全书贯穿的**教学性虚拟案例** (非真实商业体)。本示例 S1-S4/W1-W5/O1-O4/T1-T5 内容**完全来自** `assets/_raw/books/0to1-brand/ch05-self-swot.md` L411-470 原文 (Seven 在 §5.3 完整给出的"植愈坊 SWOT 分析表示例"), TOWS 战略组合的"SO/WO/ST/WT"四类策略来自 §5.4 L538-580 原文。3-6 个月优先事项是 Claude 基于该原文综合得出, 不能包装成"真实品牌成功案例"。

```
S: S1 创始人芳疗师专业 / S2 设计能力 / S3 原料严格把控 / S4 小团队灵活
W: W1 缺营销经验 / W2 产能有限 / W3 资金紧张 / W4 知名度零 / W5 SKU 单一
O: O1 天然护肤增长 / O2 疗愈体验需求 / O3 小红书种草渠道 / O4 竞品同质化
T: T1 同质化价格战 / T2 巨头跨界 / T3 偏好快变 / T4 平台流量贵 / T5 监管收紧
```

**TOWS 战略组合**:
- **SO**: 用 S1+S2 抓 O1+O2 — "芳疗师 × 设计 × 疗愈体验"切入天然护肤
- **WO**: 用 O3 弥补 W1+W4 — 小红书 KOC 种草解决营销 + 知名度
- **ST**: 用 S3 应对 T2 — 以"原料严格把控"建立巨头难以复制的工艺壁垒
- **WT**: 控制 W3 + 应对 T4 — 不做付费投放, 全靠内容自然种草

**3-6 个月优先事项**: ① 完成首批 10 SKU + 50 单测试 ② 招募 30 个 KOC 种子用户 ③ 申请化妆品生产合规资质 ④ 锁定单一稳定原料供应商

## 常见误用

1. **"漂亮格子症"**: 罗列了 S/W/O/T 四象限就停了,不做 TOWS 交叉策略 → SWOT 的价值丢失 80%
2. **S 写成"优点列表"**: 把"团队努力""产品不错"这种不满足"相对/有价值/难模仿"三条件的点也塞进去 → 后续 SO 策略无法发力
3. **O/T 与 S/W 混淆**: 把外部环境写到内部 (e.g. "竞品太强" 应放 T 不是 W; "市场政策好" 应放 O 不是 S)
4. **静态化使用**: 一次性写完就归档,不随市场动态更新 → 小米的成功正在于 SWOT 持续迭代
5. **缺数据**: SWOT 全凭直觉拍脑袋,没有客户反馈/对标分析/数据 evidence → 分析变成"感受"
6. **忽视 WT 兜底**: 只做 SO 主攻不做 WT 防御 → 当劣势 + 威胁叠加时手忙脚乱
7. **只对自己做不对竞品做**: 单 SWOT 没有比较意义,**多 SWOT 对照** 才能找差异化机会
