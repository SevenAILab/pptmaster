# Maslow · 马斯洛需求层次

## 定义

马斯洛需求层次理论由美国心理学家 Abraham Maslow 在 1943 年提出,从人类动机角度解释需求如何影响行为。经典五层为:

1. **生理需求**: 生存、食物、睡眠、身体舒适
2. **安全需求**: 稳定、可靠、健康、风险可控
3. **社交需求**: 归属、关系、陪伴、被接纳
4. **尊重需求**: 地位、自尊、被认可、专业感
5. **自我实现需求**: 成长、创造、意义、理想自我

模型卡强调: 人在某一时期会有一种需求占主导地位,其他需求处于从属地位。品牌策略中,马斯洛不是拿来套五层标题,而是帮助判断用户购买背后的深层动机。

## 适用场景

- **用户动机洞察**: 看显性需求背后的情感/社交/自我实现动机
- **价值主张分层**: 区分功能利益、安全利益、情感利益、身份利益
- **品牌定位校验**: 判断定位是否只停留在功能层,有没有更高层心智空间
- **内容选题与口号**: 用不同需求层级决定内容语气和利益表达
- **不适合**: 机械认为低层满足后才会出现高层；现实消费常常多层需求同时存在

## 使用步骤

### 第 1 步 · 明确用户行为和显性需求

先写用户正在买什么、用什么、抱怨什么。不要一上来就套五层。

输出标准: 具体用户行为 + 显性需求。例如: "敏感肌用户购买天然手工皂,显性需求是温和清洁。"

### 第 2 步 · 映射到五层需求

把同一个行为拆成多层动机:

| 层级 | 问法 | 手工皂示例 |
|---|---|---|
| 生理 | 基本功能是什么? | 清洁皮肤 |
| 安全 | 用户怕什么风险? | 怕刺激、怕假天然 |
| 社交 | 关系和归属是什么? | 送礼体面、社群同好 |
| 尊重 | 用户希望被怎么看? | 有品味、懂生活 |
| 自我实现 | 用户想成为谁? | 更会照顾自己、更有仪式感 |

### 第 3 步 · 找主导需求,不要平均用力

同一个产品可以触达多层,但策略必须判断当前主导层级。对新品牌而言,常见路径是: 先满足安全/功能可信,再放大情感/尊重/自我实现。

输出标准: 主导需求层级 + 次级需求层级 + 不优先的层级。

### 第 4 步 · 转译为价值主张和 RTB

需求层级必须落到可感知利益:
- 安全需求 → 成分公开、检测、资质、售后保障
- 社交需求 → 社群、用户故事、礼品场景
- 尊重需求 → 设计感、专业背书、身份表达
- 自我实现 → 成长叙事、创作自由、生活方式愿景

### 第 5 步 · 校验内容和产品是否同层一致

如果用户主导需求是安全,内容却一直讲高级审美,就会转化不动。反过来,如果品牌想卖尊重和自我实现,产品体验却连基本可靠都做不到,高层叙事会塌。

## 输入输出示例

### 示例 1 · 植愈坊需求层次分析 (247 书贯穿教学案例)

> **案例可追溯报告**: 植愈坊是 Seven 247 书全书贯穿的**教学性虚拟案例** (非真实商业体)。具体出处行号待 Codex 后续按 P0-4 模板填充, 当前仅标注教学属性。**禁止包装成"真实品牌成功案例"**。

| 层级 | 用户动机 | 品牌表达/动作 |
|---|---|---|
| 生理 | 温和清洁,肤感舒适 | 皂体配方、起泡、洁净力说明 |
| 安全 | 怕敏感、怕添加、怕香精刺激 | 成分公开、精油来源、敏感肌测试 |
| 社交 | 想送出有心意的礼物 | 礼盒、故事卡、节日限定 |
| 尊重 | 希望被认为有品味、懂生活 | 极简包装、芳疗师专业背书 |
| 自我实现 | 想在忙碌生活中照顾自己 | "15 分钟疗愈仪式"内容体系 |

**关键洞察**: 植愈坊不能只卖"清洁",也不能跳过安全直接卖"疗愈生活方式"。最佳路径是安全可信打底,疗愈与品味做溢价。

### 示例 2 · 苹果设计哲学 (Maslow 高层级满足)

> **案例可追溯报告**: 行业公认经典案例 (业内通用品牌定位分析对象)。本示例**只描述 Maslow 需求层次与品牌定位的映射逻辑**, 不引用具体调研数据/销售数据。苹果定位在 Maslow 高层级是业内品牌教学的经典对照, 业内人尽皆知。

Maslow 需求层次理论 (1943) 把人类需求分 5 层。苹果是高层级品牌定位的经典案例:

| Maslow 层级 | 需求类型 | 大多数品牌 vs 苹果 |
|---|---|---|
| **生理** | 吃喝睡 | 食品/饮料/快消品 |
| **安全** | 健康/安全/财产保护 | 保险/家居安防 |
| **归属** | 社交/爱/接纳 | 社交网络/社群品牌 |
| **尊重** | 地位/成就/认可 | **苹果产品 (拥有 = 一定阶层象征)** |
| **自我实现** | 创造/潜能发挥 | **苹果叙事 ("Think Different" / 让创作者成为更好的自己)** |

**关键洞察**: Maslow 的**品牌战略价值在于"定位越高层级, 价格带可越高"** — 苹果定位在尊重+自我实现层, 所以可以卖 iPhone 万元起。但有个铁律: **必须先满足下层才能讲上层** — 苹果 iPhone 必须先做到"打电话稳定"(生理) + "数据安全"(安全), 才能让"Think Different"(自我实现) 的叙事成立。否则就是"产品都做不好, 还谈什么情怀"的笑话。

## 常见误用

1. **机械金字塔**: 以为用户必须从低到高逐层满足 → 实际消费常多层并存
2. **只套标题不做洞察**: 五层表格填满,但没有解释真实购买动机 → 变成装饰
3. **跳过低层基础**: 产品安全/功能没证明,却大讲自我实现 → 用户不信
4. **所有品牌都讲高层需求**: 不是所有品类都适合卖意义感 → 过度拔高会虚
5. **忽略主导需求**: 每层都写一点 → 定位和内容没有重点
6. **不连接 RTB**: 需求层级没有证据支撑 → 只是情绪话术
7. **把需求层级等同用户等级**: 不是高收入就一定追求高层需求,也不是低价用户只看生理需求
