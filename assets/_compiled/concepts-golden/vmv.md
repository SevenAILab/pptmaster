# VMV · 愿景使命价值观

## 定义

VMV 是 Vision、Mission、Values 的缩写,中文是愿景、使命、价值观。它回答三个问题: 品牌想成为什么,品牌为什么存在,品牌相信什么。

Seven 247 书将 VMV 称为品牌灵魂: 在研究目标市场和定位策略之前,先回答品牌为何存在、最终想成为什么、信仰什么。VMV 不是墙上的空洞口号,而是品牌战略的最高指引和精神内核。它定义终极追求和行为准则,让定位选择、产品策略、传播表达和年度规划都有原点。

VMV 三者的关系是: 愿景定义远方,使命定义当下存在价值,价值观定义做事边界。好 VMV 应该真实、清晰、能指导取舍。

## 适用场景

- **品牌定位原点**: Sub-Agent ④ 在定位前先校准品牌为何存在
- **品牌屋战略层**: Sub-Agent ⑤ 将 VMV 作为品牌屋顶层价值和行为准则
- **组织共识与决策取舍**: 产品、渠道、内容、合作伙伴选择都可回到 VMV 校验
- **年度规划校准**: Sub-Agent ⑥ 检查年度 OKR 是否服务长期愿景
- **不适合**: 为了 PPT 好看而编高大上口号；如果不能指导取舍,VMV 就没有战略价值

## 使用步骤

### 第 1 步 · 收集战略输入

准备品牌定位陈述、核心价值主张、核心优势、创始人初心、目标用户价值观、行业问题和长期机会。

输出标准: 一页输入表,每项都要有证据或原话。

### 第 2 步 · 写愿景 Vision

愿景回答: 我们最终想成为什么? 想创造怎样的未来状态? 它应当鼓舞人心,但不能脱离能力和行业方向。

输出标准: 1 句远景表达 + 3 个关键词 + 1 条边界说明。

### 第 3 步 · 写使命 Mission

使命回答: 我们为什么存在? 现在为谁提供什么核心价值? 它比愿景更贴近当前业务和用户。

输出标准: 服务对象 + 核心价值 + 解决的问题。

### 第 4 步 · 写价值观 Values

价值观回答: 我们相信什么? 面对取舍时坚持什么? 价值观必须能约束行为,不能只写"诚信、创新、责任"这种通用词。

输出标准: 3-5 条价值观,每条包含行为解释和反向禁区。

### 第 5 步 · 做一致性检验

检查 VMV 是否与定位、价值主张、产品体验、品牌人格和商业模式一致。价值观是实现使命、达成愿景的行为保障。

输出标准: VMV → 定位 → 产品 → 传播的链路表。

### 第 6 步 · 转化为决策原则

把 VMV 变成可执行的选择标准。例如合作伙伴选择、产品线扩张、内容语气、用户服务标准和年度目标优先级。

输出标准: 5-8 条 "当遇到 X,我们优先 Y,不做 Z" 的原则。

## 输入输出示例

### 示例 1 · 植愈坊 VMV (247 书贯穿教学案例)

> **案例可追溯报告**: 植愈坊是 Seven 247 书全书贯穿的**教学性虚拟案例** (非真实商业体)。具体出处行号待 Codex 后续按 P0-4 模板填充, 当前仅标注教学属性。**禁止包装成"真实品牌成功案例"**。

背景: 植愈坊以芳疗手工皂切入天然护理与情绪疗愈,服务压力型都市人群。

| 模块 | 输出 |
|---|---|
| Vision | 让自然疗愈成为都市人的日常生活方式 |
| Mission | 用专业芳疗、天然配方和克制美学,为压力中的人提供可负担的身心滋养 |
| Values | 天然不夸大、专业有温度、审美服务体验、长期主义、真诚关怀 |

决策原则:

```text
如果某个爆款成分有流量但证据不足,宁愿不追热点。
如果一个包装更吸睛但制造浪费明显,优先选择更克制可持续的方案。
```

**关键洞察**: 植愈坊的 VMV 不能只说 "做天然好产品",必须把芳疗专业、情绪疗愈和审美体验连接起来。

### 示例 2 · SmallRig VMV

> **案例可追溯报告**: ⚠️ **待人工校对** — 本示例的 SmallRig 内容需逐字与 `assets/_raw/cases/标杆案例/smallrig/*.md` 真实页对照。如案例 PDF 中无对应内容, 必须按 P0-6 4 层 Fallback 改用 Tier 2 经典案例或 Tier 3 抽象示例, 禁止 LLM 编造 SmallRig 案例。

背景: SmallRig MI 升级案例中,品牌使命愿景围绕影像创作者和创作自由展开。

| 模块 | 输出 |
|---|---|
| Mission | Beyond Image Limits / 突破影像边界,让拍摄更自由 |
| Vision | Make Dreams Come True / 每个人都能实现创作梦想 |
| Values 倾向 | 共创、可靠、效率、开放、支持创作者 |

落地原则:

```text
产品开发优先响应真实创作者工作流。
品牌表达少讲自我炫技,多讲如何帮创作者完成作品。
```

**关键洞察**: SmallRig 的 VMV 将 "工具制造商" 升维为 "创作者梦想的支持系统"。

## 常见误用

1. **口号化**: 写成 "成为世界一流品牌" → 无法指导任何具体决策
2. **愿景使命混淆**: 愿景、使命都在说同一句话 → 远方和当下没有区分
3. **价值观通用词堆砌**: 诚信、创新、责任到处都能用 → 没有品牌独特性
4. **不绑定业务选择**: VMV 和产品、渠道、内容没有关系 → 只是 PPT 装饰
5. **过度拔高**: 小品牌一上来改变世界 → 用户不信,团队也无感
6. **只写正向不写禁区**: 没有说明不做什么 → 遇到诱惑时无法取舍
7. **不随阶段复盘**: 品牌阶段变化后仍沿用早期表述 → 战略原点变模糊
