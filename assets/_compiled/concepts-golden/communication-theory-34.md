# 34 个传播学理论 · Communication Theory Toolkit

## 定义

34 个传播学理论是一个传播策略选择库,用于帮助策略 Agent 判断: 说什么、对谁说、通过什么媒介说、用户如何理解、信息如何扩散、风险如何控制。

Seven 本地方法论强调,传播不是简单输出信息,而是影响注意力、理解方式、社会认同和行为决策。这 34 个理论覆盖议程设置、编码解码、叙事理论、把关人、基模、沉默的螺旋、拉斯维尔 5W、使用与满足、两级传播、社会认同、框架依赖、权威效应等。

在 PPTAgent 中,它不是给用户展示的理论百科,而是 Agent 内部的选择器: 每次传播任务只选最适合的 3-5 个理论,并转成 PPT 页面、内容动作和验证证据。

## 适用场景

- **传播策略设计**: 为 Campaign 选择议题、叙事、扩散路径和证据
- **内容策略生成**: 判断用户为什么愿意看、是否好理解、是否愿意分享
- **舆论和口碑管理**: 预判误读、极化、反噬和意见领袖扩散
- **品牌心智建设**: Sub-Agent ⑤ 将定位、口号、人格转成传播机制
- **不适合**: 把 34 个理论全部罗列进 PPT；理论必须转成动作,否则只是知识堆砌

## 使用步骤

### 第 1 步 · 明确传播任务

先回答本次传播要改变什么认知或行为: 提升知名度、解释新品、建立信任、促成试用、制造讨论,还是处理负面舆情。

输出标准: 传播目标 + 目标人群 + 当前认知障碍。

### 第 2 步 · 选择理论组

根据任务选择 3-5 个理论,不要贪多:

| 任务 | 优先理论 |
|---|---|
| 做传播策略 | 议程设置、叙事理论、框架依赖、两级传播、社会认同 |
| 做内容策略 | 编码解码、基模理论、认知负荷、使用与满足、身份表演 |
| 做舆论口碑 | 沉默的螺旋、把关人、回音室效应、第三人效果、沟通适应 |
| 做品牌心智 | 符号互动、培养理论、社会比较、权威效应、框架依赖 |

### 第 3 步 · 转成策略动作

每个理论都要回答三件事: 为什么适用、怎么转成策略动作、适合做哪一页 PPT。

输出标准: 理论 → 策略动作 → PPT 页 → 证据需求。

### 第 4 步 · 设计内容结构

围绕选定理论设计议题、故事、框架、符号、证据和 CTA。理论不是页眉,而是内容背后的机制。

输出标准: 核心信息、叙事框架、符号素材、证据清单。

### 第 5 步 · 设计扩散路径

从种子用户到大众传播,标出 KOL/KOC、平台把关机制、圈层路径和二次创作触发点。

输出标准: 扩散路径图 + 种子节点 + 平台选择。

### 第 6 步 · 预判风险并验证

检查编码解码偏差、刻板印象、信息茧房、回音室和舆论反噬。为每个关键假设设置验证数据。

输出标准: 风险清单 + 验证指标 + 预案。

## 输入输出示例

### 示例 1 · 植愈坊小红书传播策略 (247 书贯穿教学案例)

> **案例可追溯报告**: 植愈坊是 Seven 247 书全书贯穿的**教学性虚拟案例** (非真实商业体)。具体出处行号待 Codex 后续按 P0-4 模板填充, 当前仅标注教学属性。**禁止包装成"真实品牌成功案例"**。

背景: 植愈坊希望建立 "天然芳疗手工皂 = 每日情绪疗愈仪式" 的认知。

| 理论 | 策略动作 | PPT 页 |
|---|---|---|
| 议程设置 | 先占领 "下班后自我疗愈" 议题,而不是直接卖皂 | 传播主议题页 |
| 使用与满足 | 内容满足放松、审美、护肤知识和自我照顾需求 | 内容栏目页 |
| 编码解码 | 避免用户把 "芳疗" 解读成功效夸大,用温和体验表达 | 语言规范页 |
| 两级传播 | 先找敏感肌/芳疗/生活方式 KOC 做真实体验 | 扩散路径页 |
| 社会认同 | 用真实浴室仪式和用户故事降低尝试门槛 | UGC 机制页 |

**关键洞察**: 植愈坊传播不是 "产品好天然",而是让用户愿意把它当成自己的放松仪式来分享。

### 示例 2 · SmallRig 创作者共创传播策略

> **案例可追溯报告**: ⚠️ **待人工校对** — 本示例的 SmallRig 内容需逐字与 `assets/_raw/cases/标杆案例/smallrig/*.md` 真实页对照。如案例 PDF 中无对应内容, 必须按 P0-6 4 层 Fallback 改用 Tier 2 经典案例或 Tier 3 抽象示例, 禁止 LLM 编造 SmallRig 案例。

背景: SmallRig 希望强化 "支持创作者完成作品" 的品牌主张。

| 理论 | 策略动作 | PPT 页 |
|---|---|---|
| 符号互动 | 将相机笼、快装结构、创作者现场变成品牌符号 | 品牌符号页 |
| 两级传播 | 通过专业创作者和测评 KOL 影响更广泛用户 | KOL/KOC 页 |
| 框架依赖 | 从 "便宜配件" 重新框定为 "创作者工作流工具" | 叙事框架页 |
| 权威效应 | 用专业作品、导演/摄影师背书和真实项目案例建立信任 | RTB 证据页 |
| 螺旋式传播 | 新机型适配、共创故事、工作流教程连续发布 | 节奏页 |

**关键洞察**: SmallRig 的传播要让产品成为创作者身份和工作方式的一部分,而不是单次功能广告。

## 常见误用

1. **理论百科化**: 34 个全列出来 → PPT 变成课堂笔记
2. **只引用不转化**: 写了 "议程设置" 但没有议题动作 → 理论没有策略价值
3. **理论与任务错配**: 做新品种草却选一堆舆论危机理论 → 方向跑偏
4. **忽略编码解码**: 品牌想表达高级,用户却理解成贵和远 → 传播偏差未被检查
5. **只设计内容不设计扩散**: 内容好但没有种子节点和平台机制 → 难以传播
6. **忽略风险控制**: 使用刻板印象、身份表演等理论时不设边界 → 容易反噬
7. **没有验证证据**: 不设阅读、互动、转化、情绪和舆论指标 → 无法复盘
