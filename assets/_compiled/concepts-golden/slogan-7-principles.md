# Slogan 7 Principles · 品牌口号七原则

## 定义

品牌口号 (Slogan) 是品牌面向用户高频传播的一句话,用于压缩品牌定位、价值主张和情绪召唤。247 书强调,好 Slogan 能传递核心价值、引发情感共鸣、鼓励行动并增强记忆。

本黄金版将 247 书标准与 Seven 方法论整合为 7 条原则:

1. **简洁有力**: 短、顺、好念、好记
2. **聚焦核心**: 紧扣品牌定位和价值主张
3. **用户利益**: 说用户能得到什么,不是企业会什么
4. **情感共鸣**: 触动目标用户的渴望或痛点
5. **独特识别**: 和竞品拉开语言差异
6. **品牌调性**: 符合品牌人格和语气
7. **可传播可资产化**: 能长期使用,形成品牌记忆资产

## 适用场景

- **品牌建设必检**: Sub-Agent ⑤ 输出品牌屋心智层的品牌口号
- **定位表达压缩**: 将 Brand-Positioning-Triangle 和 VPC 转成用户语言
- **传播活动主线**: 为 IMC、详情页、社媒内容提供统一语言钉
- **品牌升级/再定位**: 用新口号重塑用户心智
- **不适合**: 在定位和价值主张未定前先玩文字；口号不是文字游戏

## 使用步骤

### 第 1 步 · 回到定位和价值主张

先拿到 target、category、key benefit、RTB 和核心价值主张。口号必须来自战略,不能脱离定位单独创意。

输出标准: 1 条定位陈述 + 1 条价值主张 + 3 个关键词。

### 第 2 步 · 生成多种表达方向

247 书建议尝试不同风格:
- 利益点型: 强调能带来什么好处
- 情感共鸣型: 触动内心感受
- 态度理念型: 表达品牌主张
- 行动召唤型: 鼓励用户行动

每类先生成 5-10 条,不要一开始就收窄。

### 第 3 步 · 用 7 原则筛选

给每条口号按 1-5 分评估:

| 原则 | 评估问题 |
|---|---|
| 简洁有力 | 是否足够短、好读、好记? |
| 聚焦核心 | 是否回到定位主张? |
| 用户利益 | 用户是否听得懂对自己有什么用? |
| 情感共鸣 | 是否触动目标人群? |
| 独特识别 | 是否不像竞品通用话术? |
| 品牌调性 | 是否符合人格和语气? |
| 资产化 | 是否可长期使用、可视觉化、可复述? |

### 第 4 步 · 做用户和内部小测试

247 书提醒最终选择要结合策略思考、用户感知和法律风险。至少做 3 类检查:
- 目标用户记忆度/好感度
- 内部销售/客服是否会自然复述
- 商标、竞品话术和敏感词风险

### 第 5 步 · 绑定视觉锤和应用场景

口号不是孤立文字。需要和视觉锤、主视觉、包装、详情页、社媒标题、TVC 结尾形成一致重复。

## 输入输出示例

### 示例 1 · 植愈坊口号筛选 (247 书贯穿教学案例)

> **案例可追溯报告**: 植愈坊是 Seven 247 书全书贯穿的**教学性虚拟案例** (非真实商业体)。具体出处行号待 Codex 后续按 P0-4 模板填充, 当前仅标注教学属性。**禁止包装成"真实品牌成功案例"**。

输入: 定位为"情绪疗愈洁肤",目标用户是注重生活仪式感的都市年轻女性。

| 候选 | 类型 | 评估 |
|---|---|---|
| 洗一次,疗愈一次 | 利益+情感 | 简洁、聚焦、可复述,与洗澡场景强绑定 |
| 给敏感肌一场温柔晚安 | 情感 | 适合夜间 SKU,但主品牌略窄 |
| 天然植萃,温和治愈每寸肌肤 | 利益 | 信息完整但偏长、同质化 |
| 你的私人疗愈时刻 | 情感 | 有共鸣,但品类和动作不够明确 |

**推荐**: "洗一次,疗愈一次"。它把功能动作"洗"和情绪收益"疗愈"绑定,短、顺、易记,也方便做视觉锤。

### 示例 2 · Apple "Think Different"

> **案例可追溯报告**: ⚠️ **待人工分类** — 请按 P0-6 4 层 Fallback 决策树标注本案例性质 (Tier 1 真实案例 / Tier 2 行业经典 / Tier 3 抽象示例 / Tier 4 教学案例), 并补全 source 路径 + 行号。

背景: 乔布斯回归后的苹果需要重建挑战者与创造者精神。

| 原则 | 表现 |
|---|---|
| 简洁有力 | 两个词,极短 |
| 聚焦核心 | 指向苹果与传统计算机巨头的精神差异 |
| 情感共鸣 | 呼唤不墨守成规、有创造力的人 |
| 品牌调性 | 挑战者、创造者、反主流 |
| 资产化 | 可延展为广告战役、人物海报、品牌复兴叙事 |

**关键洞察**: 好口号能超越产品本身,连接时代精神和用户身份认同。

## 常见误用

1. **先写口号后定战略**: 文字好听但没有定位支撑 → 传播无法沉淀
2. **企业视角太重**: "匠心品质,创新未来"这类话用户无感
3. **信息塞太满**: 想把所有卖点放进一句话 → 不好记也不好传播
4. **只有情绪没有利益**: 很美但不知道卖什么、解决什么
5. **过度同质化**: "让生活更美好""重新定义体验" → 任何品牌都能用
6. **不符合调性**: 温柔品牌突然热血,专业品牌突然卖萌 → 人格断裂
7. **不做测试和法务检查**: 用户不记得、商标不可用或与竞品撞车
