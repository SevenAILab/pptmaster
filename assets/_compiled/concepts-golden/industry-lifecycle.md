# Industry Lifecycle · 行业生命周期

## 定义

行业生命周期是判断一个行业从出现、增长、成熟到衰退或变革所处阶段的分析模型。常见阶段包括导入期、成长期、成熟期、衰退期,也可结合 S-Curve 表达行业增长速度的变化。

247 书强调: 判断行业处于哪个阶段,能帮助我们理解未来增长空间、当前竞争焦点和适合的策略。一个成长期行业要抢速度和心智,一个成熟期行业要拼效率、差异化和成本结构,两者打法完全不同。

## 适用场景

- **行业分析必检**: Sub-Agent ② 判断行业处于萌芽、成长、成熟还是衰退/变革
- **进入机会判断**: 决定是快速卡位、聚焦细分、还是寻找替代技术窗口
- **竞争策略选择**: 不同阶段的竞争焦点不同,对应不同资源配置
- **年度规划节奏**: 渠道、产品线和预算节奏要匹配品牌与行业阶段
- **不适合**: 用单一指标下结论；生命周期要综合增长率、竞争格局、技术成熟度、用户渗透等证据

## 使用步骤

### 第 1 步 · 明确行业边界

先定义你分析的是大行业、子行业还是新兴品类。比如"日化行业"太大,"中国线上天然手工皂市场"更适合植愈坊。

输出标准: 行业名称 + 地域 + 品类范围 + 上下游边界。

### 第 2 步 · 收集阶段判断指标

| 指标 | 观察问题 |
|---|---|
| 市场增长 | 增速是加快、放缓还是停滞? |
| 用户渗透 | 用户认知和购买是否普及? |
| 竞争格局 | 新玩家涌入、头部集中还是出清? |
| 技术/产品 | 技术是否成熟,创新是颠覆还是改良? |
| 渠道 | 渠道红利是否仍在,获客成本是否上升? |
| 利润 | 毛利和价格战趋势如何? |

### 第 3 步 · 对照阶段特征

| 阶段 | 典型特征 | 策略重心 |
|---|---|---|
| 导入期 | 认知低、技术不稳、教育成本高 | 教育市场、验证需求、聚焦种子用户 |
| 成长期 | 增速快、玩家进入、需求扩散 | 快速卡位、抢渠道、建立品类心智 |
| 成熟期 | 增长放缓、竞争激烈、效率重要 | 差异化、运营效率、品牌资产、细分创新 |
| 衰退/变革期 | 需求下降或被替代 | 转型、收缩、寻找新曲线 |

### 第 4 步 · 画 S-Curve 并标注当前位置

247 书建议可画一条 S 型曲线,把当前行业位置标在曲线上。图不是为了好看,而是为了让团队对"增长空间还有多大"形成共识。

输出标准: 当前阶段 + 判断证据 + 不确定性。

### 第 5 步 · 推导策略含义

最后必须回答:
- 这个阶段最大的机会是什么?
- 最大风险是什么?
- 品牌应采取扩张、聚焦、效率、转型中的哪种策略?
- 年度目标和渠道组合是否匹配阶段?

## 输入输出示例

### 示例 1 · 中国线上天然手工皂市场

> **案例可追溯报告**: ⚠️ **待人工分类** — 请按 P0-6 4 层 Fallback 决策树标注本案例性质 (Tier 1 真实案例 / Tier 2 行业经典 / Tier 3 抽象示例 / Tier 4 教学案例), 并补全 source 路径 + 行号。

| 判断项 | 观察 |
|---|---|
| 行业边界 | 线上天然/精油/手工皂,属于洗护与天然护肤交叉细分 |
| 阶段判断 | 成长期偏早,但部分基础手工皂卖点已同质化 |
| 证据 | 天然成分、情绪疗愈、小红书种草仍有增长；但低价手工皂竞争已多 |
| 竞争焦点 | 从"天然手作"转向"具体人群 + 情绪场景 + 合规可信" |
| 策略含义 | 植愈坊应聚焦"情绪疗愈洁肤"细分,抢新品类心智,不要做泛手工皂 |

**关键洞察**: 行业不是完全蓝海,但细分需求仍在成长。机会在于重新定义子品类,而不是进入手工皂低价红海。

### 示例 2 · 智能手机行业 (Industry Lifecycle 教科书案例)

> **案例可追溯报告**: 行业公认经典案例 (业内通用产业生命周期分析对象)。本示例**只描述行业生命周期 4 阶段的特征映射**, 不引用具体出货量/市场份额/财报数据。智能手机行业 2007-2025 的生命周期阶段是业内公开教学对象, 业内人尽皆知。

智能手机行业从 2007 年 iPhone 发布至今, 完整经历了产业生命周期 4 阶段:

| 阶段 | 时期 | 特征 | 关键事件 |
|---|---|---|---|
| **导入期** | 2007-2010 | 早期采用者, 教育市场 | iPhone 发布, 安卓兴起, 智能手机概念被验证 |
| **成长期** | 2011-2015 | 爆发性增长, 玩家涌入 | 大量厂商进入 (HTC/三星/小米等), 出货量快速增长 |
| **成熟期** | 2016-2022 | 增长放缓, 集中度提升 | 前 5 大厂商垄断, 创新周期变长, 价格战激烈 |
| **衰退/重启** | 2023+ | 总量下滑, 等待新刺激 | 出货量持续下滑, AI 手机/折叠屏寻找新增长曲线 |

**关键洞察**: Industry Lifecycle 的**战略价值在于判断"应该用什么策略"** — 导入期投教育市场, 成长期抢份额, 成熟期建壁垒, 衰退期要么收割现金要么找新曲线 (如苹果切 Vision Pro / 服务收入)。**用错阶段的策略 = 战略错位** (e.g. 在成熟期还用导入期的烧钱推广策略 = 浪费资源)。

## 常见误用

1. **行业边界过大**: 用大行业阶段替代细分市场阶段 → 结论太粗
2. **只看增长率**: 增长快不一定早期,也可能是成熟行业里的新渠道红利
3. **阶段判断没有证据**: 直接说"处于成长期" → 无法被审计
4. **忽略区域差异**: 全球成熟,中国可能成长；一线成熟,下沉可能成长
5. **把产品生命周期等同品牌生命周期**: 产品、品类、品牌可能处于不同阶段
6. **阶段与策略不匹配**: 成熟期还用烧钱教育市场,导入期却追求规模利润
7. **一次判断不更新**: 行业阶段会因技术、政策、渠道变化而迁移
