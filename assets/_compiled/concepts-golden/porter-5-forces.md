# Porter Five Forces · 波特五力竞争模型

## 定义

波特五力竞争模型由 Michael Porter 提出,用于分析一个行业的结构性吸引力和盈利压力。五种力量分别是:

1. **现有竞争者竞争强度**
2. **潜在进入者威胁**
3. **替代品威胁**
4. **供应商议价能力**
5. **购买者议价能力**

模型卡强调: 企业间直接竞争通常是五力中最显性的力量,但行业利润并不只由直接竞品决定。新进入者、替代方案、上下游议价权同样会持续挤压利润。

## 适用场景

- **行业结构分析**: Sub-Agent ② 判断行业是否值得进入、利润是否可持续
- **竞争压力解释**: 看竞争强度来自同业、替代品、进入者还是上下游
- **定位避让与切入**: 找到低压细分或可建立壁垒的位置
- **商业模式校验**: 检查 BMC 的成本、伙伴、渠道和收入是否承受五力压力
- **不适合**: 直接替代用户洞察或品牌差异化设计；五力说明行业压力,不自动给出品牌定位

## 使用步骤

### 第 1 步 · 明确行业边界和分析对象

波特五力对边界极其敏感。先确定分析的是"中国线上天然手工皂",还是"中国护肤品",还是"全球摄影配件"。边界不同,五力结论会完全不同。

输出标准: 行业边界 + 地域 + 目标客户 + 关键产品/服务范围。

### 第 2 步 · 分析现有竞争者竞争强度

看玩家数量、集中度、增长速度、差异化程度、退出壁垒和价格战情况。

判断问题:
- 头部是否集中?
- 产品差异是否明显?
- 市场增长是否足以容纳新玩家?
- 竞争主要拼价格、渠道、品牌还是技术?

### 第 3 步 · 分析进入者与替代品

**潜在进入者**看进入壁垒: 资本、渠道、技术、品牌、合规、规模优势。

**替代品**看用户完成同一 JTBD 的其他方式。替代品不一定长得像你,例如植愈坊的替代品可以是沐浴露、香薰、SPA、睡前冥想内容。

### 第 4 步 · 分析供应商和购买者议价能力

供应商议价力来自原料稀缺、专利、渠道控制、切换成本。购买者议价力来自选择多、信息透明、价格敏感、集中采购能力。

输出时要说明它们如何影响毛利、价格、交付稳定性和品牌议价。

### 第 5 步 · 汇总五力强弱与战略含义

用高/中/低评分,并输出 3 类结论:
- 行业吸引力: 是否值得进入或加码
- 关键风险: 哪一股力量最危险
- 应对策略: 建壁垒、避红海、换细分、做生态、控供应链等

## 输入输出示例

### 示例 1 · 植愈坊线上天然手工皂 (247 书贯穿教学案例)

> **案例可追溯报告**: 植愈坊是 Seven 247 书全书贯穿的**教学性虚拟案例** (非真实商业体)。具体出处行号待 Codex 后续按 P0-4 模板填充, 当前仅标注教学属性。**禁止包装成"真实品牌成功案例"**。

| 五力 | 强度 | 判断 |
|---|---|---|
| 现有竞争 | 中高 | 手工皂玩家多,低价和天然卖点同质化 |
| 新进入者 | 高 | 小批量生产和内容开店门槛低,但合规会抬高门槛 |
| 替代品 | 高 | 沐浴露、洁面、香薰、SPA 都能替代部分 Job |
| 供应商 | 中 | 高品质精油与有机原料有成本和稳定性压力 |
| 购买者 | 高 | 用户选择多,对功效与价格敏感 |

**战略含义**: 植愈坊不能靠"手工天然"进入红海,必须用"敏感肌安全证据 + 情绪疗愈场景 + 设计礼品属性"降低替代威胁并建立差异化。

### 示例 2 · 中国新茶饮赛道 (Porter 5 Forces 经典)

> **案例可追溯报告**: 行业公认经典案例 (业内通用赛道分析对象)。本示例**只描述 Porter 5 Forces 在新茶饮赛道的应用结构**, 不引用具体市场份额/销售数据/财报。中国新茶饮赛道的五力分析是业内公开教学对象, 业内人尽皆知。

Michael Porter 的五力模型 (1979) 分析行业竞争结构。中国新茶饮 (喜茶 / 奈雪 / 蜜雪冰城等) 是教科书级案例:

| 五力 | 新茶饮赛道 |
|---|---|
| **现有竞争者** | 极激烈 — 喜茶/奈雪/茶颜悦色/蜜雪冰城/古茗/书亦烧仙草等数十家 |
| **潜在进入者威胁** | 高 — 进入门槛低 (一间店即可), 加盟模式快速复制 |
| **替代品威胁** | 中 — 咖啡/瓶装饮料/果汁都可替代 |
| **供应商议价力** | 低-中 — 水果/茶叶供应商分散, 头部品牌有议价权 |
| **买家议价力** | 高 — 消费者切换成本极低 (隔壁有 5 家) |

**结论**: 五力**全部不利**, 这是个**典型红海赛道**。

**关键洞察**: Porter 五力的**战略含义不是"是否进入", 而是"以什么姿态进入"** — 红海赛道里:
- **蜜雪冰城**用"极致成本+万店密度"应对五力 (5 力都不利时, 唯一的护城河是规模)
- **喜茶**用"品牌溢价+一线城市心智"应对五力 (用品类升级跳出价格战)
- **奈雪**早期想做"高端茶饮", 五力压力下 IPO 后逐步下沉

**当 5 力全部不利时, 要么做极致规模 (蜜雪), 要么做极致品牌 (早期喜茶), "中间地带"会被两头挤压**。

## 常见误用

1. **行业边界错**: 用大行业五力解释细分机会 → 结论不准
2. **只分析直接竞品**: 忽略替代品和上下游 → 看不到真正利润压力
3. **把五力当打分游戏**: 高中低没有证据 → 无法指导决策
4. **混淆品牌五力**: 本地资料另有"品牌五力模型",它讲产品力/渠道力/营销力等,不是 Porter 五力
5. **没有战略含义**: 五力表填完就结束 → 没回答进入、避让、建壁垒
6. **静态化使用**: 政策、技术、渠道变化会改变五力强弱 → 需要定期更新
7. **忽略用户 JTBD**: 替代品来自同一任务的不同解法,不理解任务就会漏替代品
