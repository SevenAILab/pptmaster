# Aaker 品牌人格 · Brand Personality

## 定义

Aaker 品牌人格是把品牌拟人化为一组稳定性格特征的模型,帮助团队回答: "如果品牌是一个人,他会怎么说话、怎么做事、给人什么感觉?"

Jennifer Aaker 在 1997 年提出品牌人格五维度框架,常见维度包括真诚、刺激、能力、精致、粗犷。Seven 247 书在品牌语言识别章节中更偏实操: 先把品牌想成一个人,再定义人格、语调和行为方式。它不是给品牌贴一个漂亮标签,而是给命名、口号、文案、客服、视觉、内容和创始人表达提供一致标准。

在 PPTAgent 中,Aaker 五维度用于建立人格坐标,12 种品牌原型用于快速讲清人格角色,最终要落到 Tone of Voice: 同一句话,这个品牌应该怎么说。

## 适用场景

- **品牌建设人格层**: Sub-Agent ⑤ 需要输出品牌人格、语调、语言风格和触点表达
- **命名与口号判断**: 检查名称、Slogan、口吻是否符合品牌性格
- **品牌屋心智层**: 将品牌定位转译为用户能感知的性格与态度
- **客服/社媒/内容语气统一**: 让不同触点说话像同一个人
- **不适合**: 在定位、TA 和价值主张未定时先选人格；人格不是先验偏好,必须服务战略

## 使用步骤

### 第 1 步 · 提炼战略输入

先收集品牌定位、核心价值主张、目标用户价值观、竞品语气和创始人初心。人格不是凭感觉选出来的,而是从战略输入里推出来。

输出标准: 3-5 个战略关键词,例如 "治愈、专业、温柔、天然、审美"。

### 第 2 步 · 选择人格坐标

用 Aaker 五维度给品牌打坐标: 真诚、刺激、能力、精致、粗犷中哪些是主导,哪些不能碰。每个维度都要说明理由和证据。

输出标准: 主人格维度 1-2 个 + 辅助维度 1 个 + 禁区维度 1-2 个。

### 第 3 步 · 选择品牌原型

参考 12 种品牌原型,选择一个主导原型和一个辅助原型。原型要能解释品牌的渴望、目标、恐惧和策略。

输出标准:

| 字段 | 输出 |
|---|---|
| 主导原型 | 例如 Caregiver / 支持者 |
| 渴望 | 品牌持续追求什么 |
| 目标 | 帮用户完成什么 |
| 恐惧 | 品牌最不应变成什么 |
| 策略 | 该人格下最自然的行动方式 |

### 第 4 步 · 定义 Tone of Voice

把人格转成语言规则。语调不是"说什么",而是"怎么说"。同一条产品信息,专业型品牌、照顾者品牌、探索者品牌说法完全不同。

输出标准: 3-5 个语调词 + 每个词的"应该说/不要说"示例。

### 第 5 步 · 制作触点表达样张

选择 3 个高频触点写样张: 官网首屏、产品详情页、客服回复、社媒笔记、危机回应。样张必须能看出人格差异。

输出标准: 每个触点 1 段示例文案 + 1 条人格说明。

### 第 6 步 · 压力测试一致性

检查人格与定位、视觉、口号、产品体验是否一致。如果品牌说自己温柔,但客服冷冰冰；说自己专业,但证据不足,人格就会变成表演。

## 输入输出示例

### 示例 1 · 植愈坊品牌人格 (247 书贯穿教学案例)

> **案例可追溯报告**: 植愈坊是 Seven 247 书全书贯穿的**教学性虚拟案例** (非真实商业体)。具体出处行号待 Codex 后续按 P0-4 模板填充, 当前仅标注教学属性。**禁止包装成"真实品牌成功案例"**。

背景: 植愈坊是面向都市压力人群的芳疗手工皂品牌,核心价值是温和清洁与情绪疗愈。

| 模块 | 输出 |
|---|---|
| Aaker 主维度 | 真诚 + 精致 |
| 主导原型 | Caregiver / 照顾者 |
| 辅助原型 | Creator / 创造者 |
| 语调 | 温柔、专业、克制、有审美 |
| 语气禁区 | 夸张功效、网红叫卖、冷硬科普 |

客服回复样张:

```text
敏感肌可以先从无香基础款开始,在手臂内侧试用 24 小时。
如果你最近皮肤状态不稳定,我们更建议先让肌肤休息,不急着更换太多产品。
```

**关键洞察**: 照顾者人格让植愈坊不是"卖香皂",而是提供一种被理解、被照顾的日常仪式。

### 示例 2 · SmallRig 品牌人格

> **案例可追溯报告**: ⚠️ **待人工校对** — 本示例的 SmallRig 内容需逐字与 `assets/_raw/cases/标杆案例/smallrig/*.md` 真实页对照。如案例 PDF 中无对应内容, 必须按 P0-6 4 层 Fallback 改用 Tier 2 经典案例或 Tier 3 抽象示例, 禁止 LLM 编造 SmallRig 案例。

背景: SmallRig 在 MI 升级案例中选择 "支持者 Upholder" 人格,服务全球影像创作者。

| 模块 | 输出 |
|---|---|
| Aaker 主维度 | 能力 + 真诚 |
| 主导原型 | Upholder / 支持者 |
| 渴望 | 保持善意,支持他人 |
| 目标 | 帮助创作者完成拍摄任务 |
| 语调 | 可靠、直接、专业、站在创作者身边 |

触点表达:

```text
每一次拍摄都有不可重来的现场。
SmallRig 用可组合、可扩展、可快速适配的工具系统,
帮助创作者把想法稳定落地。
```

**关键洞察**: 支持者人格让 SmallRig 的工程能力不只是冷硬技术,而是"为创作者托底"的品牌态度。

## 常见误用

1. **人格先于战略**: 先说"我想要高级感/年轻化"再找理由 → 人格与定位脱节
2. **多重人格**: 官网像奢侈品,小红书像搞笑号,客服像说明书 → 用户无法形成稳定印象
3. **原型当装饰**: 只选了 "Hero/Caregiver" 名字,没有渴望、目标、恐惧和行为规则
4. **语调没有样张**: 只写 "专业、温暖、真诚" → 执行团队不知道怎么落文案
5. **人格与证据冲突**: 说自己专业却没有 RTB,说自己关怀却售后僵硬 → 信任断裂
6. **只做对外表达**: 没有影响产品、服务和组织行为 → 人格停留在传播层
7. **追热点换人格**: 为了短期流量频繁切换语气 → 品牌资产无法沉淀
