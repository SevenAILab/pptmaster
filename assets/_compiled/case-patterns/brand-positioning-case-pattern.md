# Case Pattern · 品牌定位案例

## 来源定位

- 原始文件: `品牌定位案例.pptx`
- 拆解目录: `assets/_raw/cases/品牌定位/案例库/`
- 案例品牌: 贝比赋
- 适用模块: `brand_positioning`, `consumer_insight`, `industry_analysis`, `competitor_analysis`, `brand_building`

## 可复用结构

1. **Brief 与升级目标**
   - 用品牌背景、增长风口、既有资产说明为什么要做定位升级。
   - 参考: `slide-001.md`, `slide-004.md`, `slide-005.md`, `slide-011.md`。

2. **研究策略思路**
   - 以“行业 / 竞争 / 消费者 / 自身”四类证据共同回答核心问题。
   - 定位案不是先写 slogan,而是先完成诊断。
   - 参考: `slide-013.md`, `slide-032.md`。

3. **市场扫描**
   - 先说明市场趋势和外部窗口,再解释品牌为什么此刻有机会。
   - 参考: `slide-015.md` 至 `slide-018.md`。

4. **竞争态势**
   - 对竞品按流派、支撑点、定位话术和心智占位拆解,找出可进入的空位。
   - 参考: `slide-019.md` 至 `slide-024.md`。

5. **消费者画像与心智**
   - 用人群变化、购买动机、心智抽屉和“心智第一联想”解释定位语言为什么能被记住。
   - 参考: `slide-025.md` 至 `slide-031.md`, `slide-042.md`。

6. **分析结论 -> 定位表达**
   - 把行业趋势、自身优势和消费者需求收束为一句定位。
   - 贝比赋示范: “用科学方式,致力精选营养配方的高端婴幼儿奶粉” -> “科学配比高端奶粉”。
   - 参考: `slide-039.md`, `slide-041.md`, `slide-043.md`。

7. **RTB / 核心竞争力塑造**
   - 定位必须给出相信理由: 科学性、信任感、可信度、覆盖面。
   - 参考: `slide-047.md`, `slide-048.md`。

8. **传播落地与品牌屋**
   - 用口号、营销传播策略和品牌屋把定位变成可执行体系。
   - 参考: `slide-046.md`, `slide-054.md`, `slide-079.md`。

## 使用边界

- 贝比赋是虚拟 / 教学案例,可学习结构和推导链,不得冒充真实客户数据。
- 不能把奶粉行业事实迁移到其他行业；只能迁移“诊断 -> 研究 -> 定位 -> RTB -> 落地”的结构。
- 如要引用案例原文,必须在 `data_refs.source` 指向具体 slide。
