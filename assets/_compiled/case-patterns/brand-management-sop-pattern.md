# Case Pattern · 品牌管理全工作手册 SOP

## 来源定位

- 原始文件: `2024品牌管理全工作手册.pdf`
- 拆解目录: `assets/_raw/sops/`
- 适用模块: 全部 6 个 Sub-Agent,尤其 `brand_building` 与 `annual_planning`

## 可复用结构

1. **3 x 3 品牌全案体系**
   - 品牌全案由品牌战略、品牌表达、品牌沟通三大部分组成。
   - 品牌是企业给客户的承诺,需要通过战略、表达和沟通让客户理解、认可和信任。
   - 参考: `04-第一部分品牌战略.md`。

2. **品牌战略**
   - 回答三个问题: 我是谁、我要做什么事情、我如何做好这件事情。
   - 对应组织文化、经营战略、商业模式。
   - 输出应包含使命 / 价值观、愿景 / 战略目标 / 竞争战略、价值主张 / 商业模式落地。
   - 参考: `04-第一部分品牌战略.md`。

3. **品牌表达**
   - 品牌表达由品牌故事、品牌识别、品牌声音组成。
   - 好的品牌表达不是传统 VI 八股文,而是把品牌 DNA 变成目标受众愿意接受和记忆的内容。
   - 参考: `05-第二部分-品牌表达.md`。

4. **品牌沟通**
   - 品牌沟通由传播核心主题与规范、传播流程、传播审计组成。
   - 年度 / 季度 / 月度 campaign 要在同一核心主题下展开,保证不同媒体的一致性。
   - 参考: `06-第三部分-品牌沟通.md`。

5. **管理闭环**
   - 每个传播阶段完成后都要做审计: 找到做得好的、存在不足的,进入下一轮优化。
   - 这部分与 PDCA、品牌资产管理、年度规划、危机管理联动。
   - 参考: `06-第三部分-品牌沟通.md`。

## 使用边界

- SOP 是结构和管理原则,不是客户事实源。
- 使用 SOP 时应输出“应该如何组织方案”的框架,客户具体结论仍必须来自客户资料、web_search 或案例 OCR。
- 不要把 SOP 中的 Lightao 案例语境当成客户品牌事实。
