# Case Pattern · SmallRig MI 升级标杆案例

## 来源定位

- 原始文件: `斯莫格MI升级案例.pdf`
- 拆解目录: `assets/_raw/cases/标杆案例/smallrig/`
- 案例品牌: SmallRig / 斯莫格
- 适用模块: 全部 6 个 Sub-Agent

## 可复用结构

1. **项目背景与升级语境**
   - 从品牌升级提案进入,说明为什么从配件供应商升级到影像场景产品生态。
   - 参考: `page-001.md`, `page-002.md`, `page-003.md`。

2. **行业与趋势**
   - 市场分布、增长驱动、区域机会可作为历史案例证据；最新市场数字必须另走 web_search。
   - 参考: `page-018.md`, `page-019.md`, `page-020.md`, `page-022.md`。

3. **用户洞察**
   - 内容创作者大众化、一体化高效率影像解决方案、理性需求与精神需求共同支撑定位。
   - 参考: `page-036.md`, `page-037.md`, `page-038.md`, `page-039.md`, `page-041.md`, `page-053.md`。

4. **痛点 / 收益 / RTB**
   - 全生态、全场景、全兼容、快制造对应用户对适配、效率、可靠性和创作自由的需求。
   - 参考: `page-060.md` 至 `page-063.md`, `page-073.md`。

5. **竞争与心智对标**
   - 通过手机品牌、相机品牌、行业品牌、自身品牌定位描述,找到生态领导者 / 开创者位置。
   - 参考: `page-075.md`。

6. **品牌层级与落地线索**
   - 产品品牌 SmallRig、内容平台 SmallRig、Dream Rig、CO-Design、Live Life 构成层级划分。
   - 参考: `page-076.md`, `page-077.md`。

7. **品牌建设核心事实**
   - 品牌使命: 突破影像边界,让拍摄更自由。
   - 品牌愿景: 每个人都能实现创作梦想。
   - 品牌主张: FREE YOUR DREAM / 自由创想。
   - 产品口号: Rig UP。
   - RIG 命名释义: Redefinition / Imagination / Gear。
   - 品牌人群: 影像内容创作者。
   - 差异 RTB: 全生态 / 全场景 / 全兼容 / 快制造。
   - 产品系列: 相机支撑与稳定 / 储能解决方案 / 手机支撑与稳定 / 灯光与控制系统。
   - 参考: `page-109.md`, `page-112.md`, `page-124.md`, `page-125.md`。

## 使用边界

- SmallRig 是真实标杆案例。写 SmallRig 时必须回读具体 `_raw` 页,不能只引用本 pattern。
- `inputs/smallrig/summary.md` 只能作为客户输入背景,不能替代 OCR 事实源。
- 如果某个模块在 SmallRig PDF 中没有现成内容,必须标注 strategy assumption,不得伪装为 OCR 事实。
- 行业 / 竞品的最新数据必须走 web_search,不能把 PDF 中历史信息写成最新事实。
