### 横轴 5 阶段

| 阶段 | 图中目标 | 关键输出 |
|---|---|---|
| 机会洞察 | 发现真实的市场机会与用户需求 | 市场机会、用户需求、关键洞察 |
| 策略定义 | 明确品牌定位与战略，形成差异化方向 | 定位、目标人群、差异化策略 |
| 品牌打造 | 构建品牌核心资产，产品与体验落地 | 品牌名、口号、故事、价值观、VI |
| 传播触达 | 建立品牌认知与信任，获得用户选择 | 内容资产、渠道资产、传播策略 |
| 增长迭代 | 驱动业务增长，实现品牌长期价值 | 增长指标、数据资产、品牌健康度 |

### 纵轴 7 层

| 层级 | 图中角色 | 代表概念 |
|---|---|---|
| 战略层 | 看清楚，定义方向 | PESTEL、行业五力、SWOT、价值链、OGSM、品牌定位 STP |
| 品牌层 | 立得住，建立认知 | 品牌定位、价值主张、命名、定位语、个性调性、LOGO/VI、品牌故事、品牌架构 |
| 产品层 | 有价值，构建产品竞争力 | 用户痛点、产品定位、价值主张画布、商业模式、产品组合、生命周期 |
| 体验层 | 被喜欢，全触点体验 | 用户旅程地图、关键触点、服务设计、体验指标、会员与忠诚度 |
| 传播层 | 被看见，传播触达 | 传播核心信息、内容策略、渠道组合、整合营销传播、KOL/KOC、口碑管理 |
| 增长层 | 可增长，业务增长 | 北极星指标、AARRR、增长飞轮、冷启动、用户留存、私域运营、增长效率 |
| 管理层 | 有保障，组织与资产管理 | 组织架构、品牌管理制度、资产管理机制、版权合规、危机预案、复盘机制 |

### 指导原则与衡量体系

- 指导原则: 用户中心、长期价值、差异化、真诚透明、一致性。
- 衡量体系: 认知指标、体验指标、增长指标、财务指标、品牌资产指标。
- 本质陈述: 以用户价值为原点，构建差异化品牌资产，通过一致体验与传播赢得信任与选择，实现长期增长。

## 后续编译提示

- 战略层与品牌层是 Sub-Agent ④ `brand_positioning` 的高优先级视觉位置来源。
- 产品层中的 `Value-Prop-Canvas` 与商业模式相关概念需要和 247 页书 Ch07/Ch08 对齐。
- 体验层的 `User-Journey`、服务蓝图和 HEART 指标应优先进入消费者洞察 / 体验相关概念候选。
- 增长层的 `AARRR`、增长飞轮、ICE、RFM、LTV/CAC 应优先进入年度规划 / 增长类概念候选。
- 管理层主要作为 Phase 1 后续品牌管理、年度规划和组织落地的辅助资产，不作为 Task 18 第一批 60 概念的唯一依据。
